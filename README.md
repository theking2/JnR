# Neon Runner

A simple browser-based jump and run game with three levels, keyboard controls, and a PHP/SQLite leaderboard API.

## Run locally

1. Start a local PHP server from this folder:
   `php -S 127.0.0.1:8000`
2. Open `http://127.0.0.1:8000/` in a browser.

## Features
- Three progressively harder levels
- Fullscreen mode support
- Player form for first name, last name, and email
- PHP/SQLite leaderboard storage via `api.php`
